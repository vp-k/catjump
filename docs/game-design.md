## 📱 게임 개요

**장르**: Hyper-Casual / Endless Stacker  
**플랫폼**: Mobile (Android, iOS), Web  
**타겟**: 모든 연령층, 특히 캐주얼 게이머  
**플레이 시간**: 1-3분 세션

### 게임 컨셉
**통통하고 귀여운 러시안블루 고양이**가 고양이 간식캔 타워를 쌓아 올라가는 원탭 점프 게임. 간단하지만 중독성 있는 게임플레이와 다양한 수집 요소로 장기간 플레이를 유도합니다.

> [!NOTE]
> **에셋 변경 이유**: 츄르(액상)보다 **간식캔**이 2D 스프라이트로 표현하기 훨씬 쉽고, 원통형이라 쌓기 자연스러움

## 🔊 음향 디자인 가이드 (v1.3 추가)

### 사운드 이펙트 (SFX)

| 이벤트 | 사운드 | 특징 |
|--------|--------|------|
| 점프 | "뿅" 경쾌한 효과음 | 짧고 명확한 피드백 |
| Good 착지 | "톡" 부드러운 착지음 | 중간 톤 |
| Perfect 착지 | "띵!" 밝은 효과음 | 높은 톤 + 반짝임 |
| 콤보 증가 | "띵띵!" 상승 음계 | 콤보 수에 따라 피치 증가 |
| 콤보 끊김 | "뚝" 낮은 효과음 | 아쉬움 표현 |
| 코인 획득 | "찰랑" 코인 소리 | 보상 만족감 |
| 최고기록 근접 | 긴장감 있는 심장박동 | BPM 점점 증가 |
| 최고기록 달성 | "빠밤!" 팡파레 | 화려한 성취감 |
| 게임오버 | "미야옹~" 아쉬운 고양이 소리 | 귀여움 + 재시도 욕구 |

### 배경 음악 (BGM)

| 상황 | 스타일 | BPM | 특징 |
|------|--------|-----|------|
| 메인 메뉴 | 편안한 재즈 | 80-90 | 로파이 느낌 |
| 게임 플레이 (0-20층) | 경쾌한 일렉트로닉 | 110-120 | 밝고 신나는 |
| 게임 플레이 (20-40층) | 긴장감 있는 비트 | 130-140 | 점점 빨라지는 느낌 |
| 게임 플레이 (40층+) | 인텐스 신스웨이브 | 150+ | 도전적 분위기 |
| 고양이 하우스 | 아늑한 어쿠스틱 | 70-80 | 힐링 느낌 |

### 음향 피드백 원칙

- **즉각성**: 입력 후 50ms 이내 피드백
- **구분성**: 각 이벤트별 명확히 다른 사운드
- **일관성**: 같은 액션은 항상 같은 사운드
- **비침투성**: BGM이 SFX를 덮지 않도록 믹싱

## ♿ 접근성 (Accessibility)

### 색맹 모드

| 기본 색상 | 적록색맹 대체 | 청황색맹 대체 |
|-----------|---------------|---------------|
| 🟢 Perfect (녹색) | 🔵 파랑 | ⚪ 흰색 |
| 🟡 Good (노란색) | 🟠 주황 | 🟣 보라 |
| 🔴 Miss (빨강) | ⚫ 검정 + 패턴 | 🔴 유지 |
| ⭐ 황금캔 (금색) | ✨ 별 패턴 추가 | ✨ 별 패턴 추가 |

### 시각 보조

- **고대비 모드**: 배경/전경 명확히 구분
- **큰 글꼴 모드**: UI 텍스트 1.5배 확대
- **애니메이션 감소 모드**: 플래시/펄스 효과 비활성화
- **캔 테두리 강조**: 색상 외 형태로도 구분 가능

### 조작 보조

- **원핸드 모드**: 화면 어디든 탭으로 점프
- **자동 홀드**: 길게 터치 시 연속 점프
- **햅틱 피드백**: 착지 판정별 진동 강도 차별화
  - Perfect: 강한 진동 (50ms)
  - Good: 약한 진동 (30ms)
  - Miss: 긴 진동 (100ms)

### 청각 보조

- **시각적 비트 표시**: BGM 리듬에 맞춘 UI 펄스
- **자막/텍스트**: 모든 음성/효과음에 시각적 대체

## 📚 FTUE (First-Time User Experience) - 튜토리얼 (v1.3 추가)

### 튜토리얼 철학

> **핵심 원칙**: "말하지 말고 보여줘라" (Show, don't tell)
>
> 텍스트 없이 플레이하면서 자연스럽게 배우는 구조

### Wave 기반 튜토리얼 플로우

| Wave | 점프 횟수 | 학습 목표 | 시각적 가이드 | 성공 피드백 |
|------|-----------|-----------|---------------|-------------|
| **Wave 0** | 1-3 | 탭 = 점프 | 👆 손가락 애니메이션 + "Tap!" 텍스트 | ✨ 반짝임 + 짧은 진동 |
| **Wave 1** | 4-10 | Perfect 착지 | 🎯 Perfect 존 하이라이트 (초록색 영역 표시) | 🌟 "Perfect!" + 점수 팝업 |
| **Wave 2** | 11-20 | 콤보 시스템 | 🔥 콤보 카운터 강조 + "Combo!" 툴팁 | 🎉 콤보 축하 이펙트 |

### 튜토리얼 조건

| 조건 | 동작 |
|------|------|
| 첫 플레이 감지 | `localStorage.tutorialComplete === false` |
| 튜토리얼 스킵 | 5초 무입력 시 "Skip Tutorial?" 버튼 |
| 재학습 옵션 | 설정 → "튜토리얼 다시 보기" |
| 완료 기록 | Wave 3 완료 시 `tutorialComplete = true` |

### 튜토리얼 Best Practices (참고: Crossy Road, Stack)

- ✅ **즉시 플레이 가능**: 튜토리얼도 실제 게임플레이
- ✅ **실패 없음**: Wave 0-1에서는 게임오버 없음 (자동 구조)
- ✅ **점진적 난이도**: Wave별로 새 개념 1개씩만 추가
- ✅ **시각적 강조**: 텍스트보다 화살표, 하이라이트 사용
- ✅ **짧은 세션**: 30초 내 핵심 메카닉 전달

### 튜토리얼 자동 구조 시스템

| Wave | 자동 구조 | 시각 피드백 | 에너지 소모 |

## 🎮 핵심 게임 메카닉

### 1. 코어 게임플레이: Jump & Stack

#### 기본 메카닉
- **원탭 컨트롤**: 화면 탭 → 고양이 점프
- **캔 스택 방식**: 새 캔이 옆에서 밀고 들어옴 → 점프 안 하면 밀려서 떨어짐
- **타이밍 게임**: 정확한 타이밍에 점프해야 간식캔 위에 착지
- **수직 스크롤**: 카메라가 고양이를 따라 위로 이동
- **무한 타워**: 끝없이 높이 쌓을 수 있는 엔드리스 모드

#### 게임 플로우
```
1. 고양이가 캔 위에 서있음
2. 새 캔이 옆에서 밀고 들어옴
3. 탭 → 고양이 점프 (위로)
4. 새 캔이 기존 캔 위에 쌓임
5. 고양이가 새 캔 위에 착지
6. 점프 안 하면 → 캔에 밀려서 떨어짐 → 게임오버
```

### 2. 착지 판정 시스템 (v1.3 개선)

| 판정 | 범위 | 점수 | 피드백 |
|------|------|------|--------|
| **Miss** | 캔 밖 | 0 | 게임오버 |
| **Good** | 캔 가장자리 25% (좌우) | +10 | 고양이 휘청 + 작은 흔들림 |
| **Perfect!** | 캔 중심 **50%** ← (40%에서 완화) | +25 | 이펙트 + 사운드 + 화면 펄스 |

> **v1.3 변경**: Perfect 판정 영역을 40% → 50%로 완화하여 플레이 경험 개선

### 3. 콤보 시스템 (v1.3 개선 - 핵심 중독 요소)

**Perfect 연속 착지 시:**

| 연속 | 배율 | 피드백 |
|------|------|--------|
| 1회 | x1 | - |
| 2회 | x1.2 | "Nice!" |
| 3회 | x1.5 | "Great!" |
| 5회 | x2 | "Amazing!" + 고양이 표정 변화 |
| 10회 | x2.5 | "Incredible!" |
| 15회 | x3 | "LEGENDARY!" |
| 20회 | x4 | "GODLIKE!" + 특별 이펙트 |

**v1.3 핵심 변경:**

- ✅ **Good 착지 시 콤보 유지** (리셋하지 않음!) → 초보자 친화적
- ✅ **Perfect만 콤보 증가** → 숙련도 보상
- ✅ **Miss만 콤보 리셋** → 적절한 긴장감
- ✅ **배율 완화**: x5 → x4 최대 (밸런스 개선)

### 4. 특수 캔 시스템 (Phaser 최적화)

별도 장애물 없이, 캔 자체에 변화를 주어 난이도와 재미 조절

## 💰 수익화 전략 (Monetization)

> [!IMPORTANT]
> **핵심 원칙**: 광고는 **절대 유저 이탈을 시킬 정도로 넣지 않는다**
> 
> **우선순위**: 이탈 방지 > 단기 수익

### 1. 광고 (Ads) - 신중한 접근

#### Rewarded Ads (보상형 광고) - 주력
- ✅ **부활**: 게임 오버 시 광고 보고 계속하기
- ✅ **2배 보상**: 광고 보고 코인 2배
- ✅ **럭키 박스**: 광고 보고 추가 상자 획득
- ✅ **100% 선택적**: 유저가 자발적으로 봄

**장점**: 
- 유저가 선택하므로 이탈 없음
- 오히려 게임 경험 향상
- 광고 단가 높음 (선택적이라 참여도 높음)

#### Interstitial Ads (전면 광고) - 극도로 제한
- ⚠️ **빈도**: 5-7판마다 1회만
- ⚠️ **타이밍**: 게임 오버 후에만
- ⚠️ **스킵 가능**: 5초 후 닫기 가능
- ⚠️ **광고 제거 옵션**: IAP로 제거 가능

**제한 이유**:
- 너무 자주 = 유저 이탈
- 이탈 > 광고 수익 손실

#### Banner Ads - 사용 안 함
- ❌ **배너 광고 없음**
- 이유: 게임 화면 방해, UX 저해

### 2. IAP (In-App Purchase)

#### 프리미엄 패키지
- 💎 **광고 제거**: $2.99 (가장 인기 예상)
- 🎁 **스타터 팩**: $4.99
  - 코인 10,000
  - 다이아 500

| **빌드 속도** | ✅ 빠름 | 🟡 느림 |

#### 최종 결론: Capacitor가 최선

**이유**:
1. ✅ **Phaser와 완벽한 호환**: HTML5 게임을 그대로 래핑
2. ✅ **단순함**: 복잡한 브릿지 없이 WebView로 실행
3. ✅ **충분한 성능**: 2D 캐주얼 게임에 완벽

6. ✅ **코드베이스 단일화**: 웹 + 모바일 같은 코드
